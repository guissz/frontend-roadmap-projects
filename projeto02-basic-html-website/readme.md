# Basic HTML Website

## Objetivo
Criar uma página web simples utilizando apenas HTML com foco no uso correto de tags HTML.

## Requisitos
- Uso correto da semântica HTML na estrutura da página.
- Múltiplas páginas e uma navbar.
- SEO meta tags na head de cada página.
- A página Contact deve conter um formulário contendo campos como: name, e-mail, message, etc.
