# Single Page CV

## Objetivo
Criar um CV (Curriculum Vitae) utilizando apenas HTML.

## Requisitos
- **HTML Semântico**: Usar tags HTML apropriadas para estrutura do CV.
- **SEO Meta Tags**: Incluir meta tags essenciais para SEO.
- **Open Graph (OG) Tags**: Adicionar OG tags para compartilhamento de redes sociais.
- **Favicon**: Adicionar um Favicon na página do CV.
