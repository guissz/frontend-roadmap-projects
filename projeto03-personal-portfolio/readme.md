# Personal Portfolio

## Objetivo

Estilizar o projeto [Basic HTML Website](https://github.com/guissz/frontend-roadmap-projects/tree/main/projeto02-basic-html-website) utilizando CSS para criar layouts responsivos.

## Requisitos

- Página 100% estilizada, layout responsivo utilizando a mesma estrutura do projeto anterior. 
- Uso consistente de uma paleta de cores e tipografia.
- Uso apropriado de ferramentas CSS como Flexbox, Media Query, Box Model, etc.
- Navbar responsiva e formulário de contato estilizado.
